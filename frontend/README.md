# NexusIDE Frontend

React, TypeScript, Vite, and Tailwind frontend for the NexusIDE collaborative cloud IDE.

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` type-checks and creates a production build.
- `npm run lint` runs ESLint across the frontend.
- `npm run preview` serves the production build locally.

## Runtime Expectations

The app expects the backend API, WebSocket collaboration server, terminal relay, GitHub auth flow, and preview endpoint to be available on `localhost:4000`.
